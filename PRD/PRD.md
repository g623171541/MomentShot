#### 1. 产品概述
**产品定位**：一款为“瞬间捕捉”而生的轻量级相机应用，将拍照与录像融合为单一、直觉化的操作，消除模式切换的等待，实现从启动到成片的最短路径。  
**目标用户**：希望随时快拍记录生活、不愿错过突发现场，且偏好极简交互的普通用户与短视频创作者。  
**核心价值**：  
- **零等待**：打开即取景，点按即拍照，长按即录像，无需切换模式。  
- **沉浸式**：全屏取景，极简UI，关键操作触手可及。  
- **顺手可靠**：符合单手操作习惯，支持锁屏快捷启动，关键时刻不卡顿。

---

#### 2. 整体设计理念与核心交互
- **统一快门**：底部中央仅有一个大面积快门键。  
  - **点按**：立即拍照。  
  - **长按**：开始录制视频（带触觉反馈与进度指示）。  
  - **长按并上滑**：锁定录像状态，此时可松开手指，录像持续进行，直至点击停止按钮。  
- **无模式切换**：彻底取消“照片/视频”拨盘，依靠手势自然区分拍摄意图。  
- **界面分层**：取景器始终占据全屏，功能控件以半透明悬浮层呈现，最大程度保留构图视野。

#### 3. 详细功能需求

| 模块 | 功能点 | 描述与规则 |
|------|--------|-------------|
| **快速启动** | 冷启动速度 | 应用启动后直接进入取景画面，无广告、无过渡动画，目标从点击图标到可拍摄 < 0.8秒。 |
|  | 锁屏快捷入口 | 支持系统锁屏相机图标、控制中心快捷方式、双击电源键（手势依系统）直接唤醒相机。 |
|  | 后台拍摄 | 支持在手机待机状态下，通过音量键或蓝牙遥控器触发拍照/录像（设置中可开启）。 |
| **拍摄功能** | 拍照 | 单点快门键，自动完成连续对焦与测光，拍摄后**保存至 App 沙盒内部存储**（不写入系统相册），快门伴轻微动画及振动反馈（可关闭）。文件类型标记为 `photo`。 |
|  | 录像 | 长按快门键开始，取景框顶部出现红色录制指示与计时，手指松开则停止并自动保存至 App 沙盒内部存储（不写入系统相册）。若执行"上滑锁定"，界面显示红色停止按钮，需手动点击停止。文件类型标记为 `video`。 |
|  | 暂停录像 | 在锁定录像界面，提供暂停/继续按钮（可选开启），便于分段录制，最终输出单一视频文件保存至沙盒视频目录。 |
| **辅助功能** | 对焦与测光 | 点击取景器任意位置进行对焦，出现对焦框并附带曝光补偿滑杆（上下拖动小太阳图标调节）。长按对焦位置1秒锁定AE/AF，再次点击取消锁定。 |
|  | 变焦 | 双指捏合/扩展手势无极变焦（0.5x~5x），亦可在设置中开启单指滑动变焦条。 |
|  | 前后镜头切换 | 取景器右下角圆形切换按钮，点击翻转镜头，支持双击取景器空白区域快速切换（可设置）。 |
|  | 闪光灯 | 顶部工具条提供：自动/开/关/手电筒四态，点击循环切换，状态图标即时变化。 |
|  | 定时拍摄 | 支持3秒、10秒倒计时，选择后快门键显示倒计时数字，过程可取消。 |
|  | 辅助网格与水平仪 | 设置中开启九宫格参考线及实时水平仪指示，辅助构图。 |
| **内容管理** | 快速预览 | 左下角显示最近拍摄缩略图（**仅读取 App 沙盒内**最新一条记录：照片为静态首帧、视频为视频首帧并叠加 ▶ 角标）。点击缩略图进入简单浏览器：左右滑动浏览媒体，上滑删除当前项，底部提供分享按钮（系统分享菜单）及"..."更多操作。 |
|  | 类型筛选 | 浏览器顶部支持「全部 / 照片 / 视频」分段切换，依据文件类型字段过滤。 |
|  | 预览返回 | 在预览界面按住画面向下拖拽，或点击左上角返回箭头，平滑退回取景器，保证随时继续拍摄。 |
| **存储与文件管理** | 存储位置 | 所有照片与视频**全部写入 App 沙盒**（`Documents/` 目录下），**不写入系统相册**，App 卸载后随之清除。 |
|  | 类型分目录 | 按媒体类型区分子目录：`Documents/Media/Photos/` 存放照片，`Documents/Media/Videos/` 存放视频。 |
|  | 命名规则 | 文件名格式 `<type>_<yyyyMMdd_HHmmssSSS>.<ext>`：照片用 `photo_` 前缀（扩展名 `.heic` 或 `.jpg`），视频用 `video_` 前缀（扩展名 `.mp4` 或 `.mov`）。 |
|  | 元数据索引 | 维护一份本地索引（如 SQLite / Core Data / JSON），记录每条媒体的：`id`、`type`（`photo`/`video`）、`fileName`、`relativePath`、`createdAt`、`duration`（仅视频）、`width`、`height`、`fileSize`。预览/筛选/排序均基于该索引，避免每次扫描磁盘。 |
|  | 主动导出 | 仅当用户在预览页显式点击"分享/导出到相册"时，才请求系统相册写入权限并执行导出；默认流程下不申请相册权限。 |
| **设置中心** | 基础参数 | 视频分辨率(720p/1080p/4K)、帧率(30/60fps)、照片分辨率、地理标签开关。 |
|  | 音量键功能 | 定义音量键作用：拍照、录像开始/停止、变焦，或禁用。 |
|  | 静音拍摄 | 完全消除快门音（需符合地区法规） |
|  | 存储管理 | 显示沙盒内照片/视频各自数量与占用空间，提供按类型批量清理、单条删除入口。 |
|  | 界面定制 | 网格/水平仪开关、预览自动播放、手势配置等。 |

#### 4. 界面交互设计详解

##### 4.1 主取景界面
整体布局采用全屏取景 + 悬浮控件的结构，适应单手握持。

- **顶部状态栏（距安全区10pt，水平分布）**  
  - 左侧：**闪光灯图标** — 点击切换4种状态，图标分别为闪电加A、闪电、闪电加叉、灯泡。  
  - 中部：**定时器** — 显示“关/3s/10s”，选中后出现倒计时浮层。  
  - 右侧：**设置齿轮** — 点击从底部弹出半屏设置面板，透明毛玻璃效果，不阻断取景。  

- **取景器中央区域**  
  - 可点击对焦，对焦框白色动画缩放至目标位置，右侧出现**曝光滑杆**（太阳图标），单指上下滑动调节曝光补偿，数值范围-2EV~+2EV。  
  - 对焦/曝光锁定后显示黄色锁形图标。  
  - 辅助网格：三分线白色半透明叠加。  
  - 变焦：双指缩放时，底部短暂出现变焦倍率指示条。

- **左下角预览区**  
  - 一个40x40pt的圆角矩形缩略图，显示最近拍摄内容。  
  - 点击后以弹性动画展开至全屏预览；左滑缩略图可快速将当前显示的照片删除（需二次确认震动反馈）。

- **右下角镜头切换**  
  - 圆圈内有相机循环箭头图标，点击触发翻转动画，配合轻微振动。

- **底部快门区域（距底部安全区20pt）**  
  核心组件：  
  - **快门键**：位于水平居中，外径68pt，白色粗描边圆环（3pt），内部填充半透明白。  
    - 点按：圆环缩小到85%再弹回，短振动（10ms），屏幕闪白一次模拟快门。  
    - 长按（>0.3s）：外环变为红色进度圆环，从0点顺时针填充，超过0.5s后开始录像，进度环随录影时长增长，手指松开即停止录像并重置。  
    - 长按+上滑：当手指向上滑动超过30pt，红色圆环变为红色方块停止按钮，并出现提示“已锁定录制”，此时手指可离开，录制持续。要停止录制，点击该红色方块按钮。  

##### 4.2 录像状态反馈
- **录像进行中（未锁定）**：  
  - 顶部状态栏背景变为半透明红色，闪烁红点并显示录制计时（分:秒）。  
  - 快门按钮形态变为红色圆环填充增长（进度），手指离开则恢复默认。  
- **录像锁定录制**：  
  - 快门按钮彻底变为红色正方形停止按钮，中央有白色方块图标。  
  - 顶部红色条常亮，计时持续。  
  - 取景器其余按钮（闪光灯、切换镜头等）灰显不可用，防止误触。  
  - 点击停止按钮后，动画收起录制状态，缩略图立即更新为视频首帧。

##### 4.3 快速预览与内容浏览
- 点击缩略图，当前画面以全屏浮层呈现，背景模糊。  
- **浏览**：左右滑动切换照片/视频，视频自动静音循环播放前3秒。  
- **手势**：  
  - 上滑当前媒体（照片或视频）触发删除，伴随红色删除指示和“已删除”提示，支持摇动撤销。  
  - 点按画面呼出工具栏：分享、收藏、编辑（跳转系统编辑器）、信息。  
  - 从顶部向下滑动或者点击底部向上箭头图标，关闭预览回取景器。  

##### 4.4 设置界面
- 由主界面顶部齿轮图标或预览界面"更多"进入，以**底部抽屉**形式上升，覆盖取景器约60%高度，其余区域半透明可点击返回。  
- 分区列表：拍摄参数、长按设定、音量键、显示辅助、存储管理、关于。  
- 「存储管理」分区内分别列出"照片：N 张 · X MB"与"视频：N 个 · Y MB"两行，支持点击进入按类型清理。  
- 修改设置后立即生效，无需保存按钮，向左滑动即可关闭抽屉。

#### 5. 非功能需求
- **性能**：  
  - 相机冷启动时间 ≤ 800ms（主流中端机型）。  
  - 拍照延迟（从点击到提示音结束） ≤ 300ms。  
  - 录像启动延迟 ≤ 200ms。  
- **稳定性**：长时间录像不闪退，文件自动分段写入沙盒视频目录防止丢失；沙盒可用空间不足时给出明确提示并阻止继续拍摄。  
- **权限**：首次启动仅申请相机、麦克风权限（媒体落盘走 App 沙盒，无需相册写入权限）；若拒绝关键权限则提供引导浮层与去设置链接。位置、相册写入等权限按需在使用时再申请。  
- **适配**：支持iOS 15+ / Android 10+，全面屏安全区适配，横竖屏自动旋转且UI自适应。  

---

#### 6. 权限需求清单
- **相机**：必要，无此权限无法拍摄。  
- **麦克风**：录像必要，若拒绝则录像功能禁用，拍照正常。  
- **相册写入（可选 / 按需）**：仅当用户主动点击"导出到系统相册"时才请求；默认拍摄流程不依赖此权限，所有照片/视频均落入 App 沙盒。  
- **位置（可选）**：用于照片地理标记，不授权不影响拍摄。
